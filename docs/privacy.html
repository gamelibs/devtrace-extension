Privacy Policy

Effective Date: July 1, 2025

Last Updated: July 1, 2025

Summary: Web Request Capture Pro operates entirely on your device. We do not collect, store, or transmit any personal data to external servers.

1. Information We Collect

DevTrace is designed for web developers and security researchers to analyze network requests locally. Here's what the extension accesses:

  • Network Request Data: URLs, headers, and response codes of websites you choose to analyze
  • Browser Tab Information: URL and title of tabs where analysis is performed
  • User Settings: Your preferences for filtering and analysis options

2. How We Use Information

All collected information is used solely for:

  • Displaying network request analysis results to you
  • Filtering requests based on your preferences
  • Exporting analysis results to your local device
  • Maintaining your settings and preferences

3. Data Storage and Security

Important: All data processing happens entirely on your device. No information is transmitted to external servers.

  • Local Storage Only: All captured data is stored locally in your browser
  • No Cloud Sync: Data is never synchronized to external services
  • User Control: You can clear all data at any time
  • Temporary Capture: Request data is only stored while actively analyzing

4. Permissions Explanation

The extension requests the following permissions:

  • tabs: Access tab information to identify which website is being analyzed
  • webRequest: Monitor network requests (only for websites you specify)
  • webNavigation: Track navigation events for analysis
  • scripting: Inject analysis scripts into web pages
  • storage: Save your preferences locally in your browser
  • activeTab: Access the currently active tab for analysis
  • downloads: Save analysis results to your downloads folder
  • host permissions: Requested only when you specify a website to analyze

5. Data Sharing

We do not share any data with third parties. This extension:

  • Does not transmit data to external servers
  • Does not use analytics or tracking services
  • Does not share information with advertisers
  • Does not connect to any external APIs

6. Your Rights and Controls

You have complete control over your data:

  • Grant/Revoke Permissions: Control which websites the extension can analyze
  • Clear Data: Remove all captured requests and settings at any time
  • Disable Extension: Turn off or uninstall the extension whenever you want
  • Export Data: Save analysis results to your device

7. Children's Privacy

This extension is designed for web developers and technical users. We do not knowingly collect information from children under 13.

8. Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated effective date.

9. Contact Information

If you have questions about this privacy policy or the extension's data practices, please contact us:

  • Email: privacy@gamelibs.github.io
  • GitHub: Open an issue

Open Source: This extension's source code is available for review to ensure transparency in our privacy practices.
