Download & Install

üìù Manual Installation Instructions:

  1. Download the devtrace-v2.3.zip file
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked" and select the extracted folder
  6. The DevTrace extension will appear in your browser toolbar

Note: Since this extension is not yet published on Chrome Web Store, you need to install it manually using Developer mode.
